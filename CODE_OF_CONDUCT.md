# Code of Conduct

This project follows the [Contributor Covenant v2.1][cc].

In short: be respectful. Harassment, personal attacks, and discriminatory
language are not welcome. Maintainers may remove comments, close issues,
or block users for violations.

Report unacceptable behavior by opening a private GitHub Security
Advisory against this repo (Security tab → Report a vulnerability);
that inbox also reaches the maintainers confidentially.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
